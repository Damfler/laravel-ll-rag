# Company Wiki — LLM + RAG: План разработки

## Стек технологий

| Слой | Технология |
|------|-----------|
| Backend | Laravel 12 (PHP 8.4) |
| Frontend | Vue 3 + Inertia.js + TipTap |
| База данных | PostgreSQL 16 + pgvector |
| Кэш / Очереди | Redis 7 |
| Контейнеры | Docker + Docker Compose |
| LLM провайдеры | OpenAI / Anthropic / Ollama (абстракция) |
| Эмбеддинги | text-embedding-3-small (OpenAI) / Ollama |
| Поиск | Гибридный: PostgreSQL FTS + pgvector |

## Фаза 3 — RAG Pipeline

> Цель: индексация контента вики и семантический поиск.

### Chunking (разбивка на чанки)
- [ ] Сервис `PageChunker` — разбивка текста страницы на чанки
- [ ] Стратегия: Recursive Character Splitter (~500 токенов, перекрытие 50)
- [ ] Сохранение мета-данных чанка (page_id, space_id, заголовок, номер чанка)
- [ ] Обработка разных форматов (HTML → plain text для эмбеддинга)

### Embeddings (векторизация)
- [ ] Интерфейс `EmbeddingProviderInterface` (метод `embed(string $text): array`)
- [ ] Драйвер `OpenAiEmbeddingProvider` (text-embedding-3-small, 1536 dim)
- [ ] Драйвер `OllamaEmbeddingProvider` (локальные модели)
- [ ] Выбор провайдера через конфиг / `.env`

### Хранение в pgvector
- [ ] Миграция: таблица `page_chunks` (id, page_id, content, embedding vector(1536), metadata jsonb)
- [ ] Индекс HNSW для быстрого поиска (`ivfflat` или `hnsw`)
- [ ] Репозиторий `ChunkRepository` (сохранение, удаление, поиск по вектору)

### Очереди (асинхронная индексация)
- [ ] Job `IndexPageJob` — чанкинг + эмбеддинг + сохранение
- [ ] Job `DeletePageIndexJob` — удаление чанков при удалении страницы
- [ ] Event listener: автозапуск индексации при создании/обновлении страницы
- [ ] Команда `artisan wiki:reindex` — переиндексация всей вики

### Гибридный поиск
- [ ] Сервис `HybridSearchService`
- [ ] Полнотекстовый поиск (PostgreSQL FTS, BM25-like)
- [ ] Векторный поиск (cosine similarity через pgvector)
- [ ] Объединение результатов (Reciprocal Rank Fusion — RRF)
- [ ] Возврат топ-N наиболее релевантных чанков

## Ключевые соглашения

### Структура директорий (Laravel)
```
app/
├── Services/
│   ├── Llm/                  # LLM Gateway (интерфейс + драйверы)
│   ├── Rag/                  # RAG Pipeline (chunker, embedder, search)
│   └── Wiki/                 # Бизнес-логика вики
├── Jobs/                     # Очереди (IndexPageJob и др.)
├── Events/ & Listeners/      # Авто-индексация при изменениях
└── Http/
    ├── Controllers/
    └── Middleware/
```

### Переменные окружения
```
LLM_PROVIDER=openai           # openai | anthropic | ollama
OPENAI_API_KEY=...
ANTHROPIC_API_KEY=...
OLLAMA_BASE_URL=http://ollama:11434
```

### Команды разработки
```bash
make up           # Запустить Docker
make shell        # Войти в контейнер
make migrate      # Применить миграции
make artisan CMD="wiki:reindex"  # Переиндексировать вики
```
