Работаем в папке "Акции", которая находится в общем доступе ([[\\WEB\content]]\Росберг\Акции).

Выкачиваем актуальный файл [[https://docs.google.com/spreadsheets/d/1X-JbPeetIq4nTp9Mlu7YalF1hJgZ0k90r0XsorU0nU0/edit#gid=1714477396]]

Сохраняем его в [[\\WEB\content]]\Росберг\Акции\Оригинал Формируем из него файлы заливки по опту и рознице:

По **опту**:

  * **1 столбец - Артикул Сайта.** Выставить условное форматирование на повторение значений. Если при внесении новых данных подсвечивает, что такой артикул в файле есть, то ему необходимо указать последние актуальные данные по ценам и складам.
  * **2 столбец - старая ОПТовая цена**
  * **3 столбец - новая ОПТовая цена**
  * **4 столбец - ID склада (-ов), на которых есть акционные позиции.** Если таких складов несколько, указываются разными строками с разным ID

Загружаем тут: [[https://rest.mtkrosberg.ru/fixed_prices/opt/?TYPE=DOWNLOAD]]

По **рознице**: Артикул сайта/старая розница/новая розница, в файле заливки 3 столбца

Загружаем тут: [[https://rest.mtkrosberg.ru/fixed_prices/retail/?TYPE=DOWNLOAD]]

Если в файле оригинала не указаны старые цены, ставим их сами такими, как они сейчас в хоуме по данной позиции. Если сейчас на хоуме нет по этой позиции опта и/или розницы, ставим в известность постановщика задачи, и спрашиваем, какую старую цену поставить

Прогнали файлы, после финиша и публикации обязательно проверяем в **боевой таблице**, подхватилась ли нужная акционная цена по новым позициям.

**Если нужно убрать из акции**

Выкачиваем актуальный файл [[https://docs.google.com/spreadsheets/d/1X-JbPeetIq4nTp9Mlu7YalF1hJgZ0k90r0XsorU0nU0/edit#gid=1714477396]]

Проверяем, убрал ли постановщик задачи из него позицию, по которой снимаем акцию;

Сохраняем файл в [[\\WEB\content]]\Росберг\Акции\Оригинал

Убираем позицию из механизма акционных цен (жмём на крестик, на красном фоне)

**Обязательно убираем позиции, которые снимали, из файлов заливки по акциям опту и рознице!**

Финишуем, публикуем, проверяем, отписываемся о выполнении
