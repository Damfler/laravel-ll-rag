====== GOODRIDE КИТАЙ  ======

**Категории товаров**: гусеницы агро.

**Шаблонное обновление**: "GOODRIDE (Китай) - ГУСЕНИЦЫ"

**Обновляем**: цены. Остатки - статус "под заказ".

Сам прайс выглядит так:

{{:start:goodride_китай_прайс.jpg?direct |}}

Файл заливки состоит из:
  * 1-й столбец - артикул
  * 2-й столбец - Номенклатура
  * 3-й столбец - цена Вход (Цену вход нужно брать из столбца "Цена завода MRB")
  * 4-й столбец - цена оптовая (в рублях, из прайса)
  * 5-й столбец - цена розничная (в рублях, из прайса)
  * 6-й столбец - статус "O".

Выглядит файл заливки примерно так:
