<wrap em>Данный прайс присылается для двух брендов: Roadguider и ForeRunner.</wrap>

Для формирования наименования нужно добавить столбцы: **пробел**, **бренд** (Roadguider / ForeRunner).

Сцепляем столбцы (через пробел): **Бренд** (Roadguider или ForeRunner), "**Size**", "**Ply**", "**TT/TL**", "**Pattern**".

Файл заливки формируется один, для двух брендов. <wrap hi>Их отличия между собой - это бренд и наличие артикула поставщика (для Roadguider берется артикул из столбца "NEW CODE", для ForeRunner - колонка пустая, артикул формируется из наименования).</wrap>

Файл заливки выглядит примерно так:
{{:start:qindao_заливка.jpg?direct |}}

Порядок столбцов в файле заливки:
  - Артикул (для бренда Roadguider - "NEW CODE", для ForeRunner - пустой)
  - Наименование (отличается только бренд).
  - Цена вход (в валюте)
  - Цена оптовая (в рублях)
  - Цена розничная (в рублях)
  - Статус (Под заказ) "O".
