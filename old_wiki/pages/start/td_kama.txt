====== ТД Кама завод ======

**Шаблонное обновление**: "ТД Кама завод", "ТД Кама завод - наши склады камеры ОПТ Новосиб", "ТД Кама завод - наши склады камеры ОПТ Ростов"

**Обновляем**: цены и остатки

**Категории товаров**: шины грузовые, шины сх/сп, камеры, ободные ленты

Сам прайс выглядит так:
{{ :start:кама_прайс.jpg?direct&800 |}}

<wrap hi>**Заливаем шины через шаблон "ТД Кама завод".**</wrap>

В файле заливки грузовых и сх/сп шин нужно взять все комплектации*: 
  * для TTF без о.л и TL мы берем входную цену из колонки "Минимальные цены на условиях отсрочки, руб. с НДС" предварительно сложив цены "Шина" и "Шина в комплекте с камерой" (в настоящее время не встречалось позиций, которые имели бы обе цены); Артикул поставщика и номенклатуру берем из прайса. Аналогично нужно добавить оптовую цену - её берем из колонки "Рекомендованные оптовые цены (РОЦ), руб. с НДС" (она же "РОЦ" в алгоритме)
  * для TTF мы берем цену из "Шина в комплекте с камерой и ободной лентой". Колонку "Артикул" оставляем пустым, формируется из наименования, номенклатура - из прайса. Аналогично нужно добавить оптовую цену - её берем из колонки "Рекомендованные оптовые цены (РОЦ), руб. с НДС" (она же "РОЦ" в алгоритме)

По итогу в файле заливки для грузовых и сх/сп шин указываем:
  * 1-й столбец - Артикул (только для TTF без о.л и TL шин)
  * 2-й столбец - Наименование (для всех шин)
  * 3-й столбец - цена вход (написано выше как её корректно взять из прайса для всех комплектаций)*
  * 4-й столбец - цена оптовая (написано выше как её корректно взять из прайса для всех комплектаций)*
  * 5-й столбец - цена розничная (из алгоритма)
  * 6-й столбец - статус (под заказ)


Примерно файл заливки для сх/сп шин и грузовых будет выглядеть так:

  * Синей рамкой отмечены позиции с комплектацией TTF

<wrap hi>**Камеры заливаются по трём шаблонам:"ТД Кама завод", "ТД Кама завод - наши склады камеры ОПТ Новосиб", "ТД Кама завод - наши склады камеры ОПТ Ростов"**</wrap>

В файле заливки камер указываем:
  * 1-й столбец - Артикул
  * 2-й столбец - Номенклатура
  * 3-й столбец - цена вход (из колонки "Минималь-ные цены на условиях отсрочки, руб. с НДС"
  * 4-й столбец - цена оптовая (из алгоритма)
  * 5-й столбец - цена розничная (из алгоритма)
  * 6-й столбец - статус (под заказ)
  * 9-й столбец  -  ОПТ Ростов (скопировать опт из 4 столбца):!:
  * 10-й столбец - ОПТ Новосиб (скопировать опт из 4 столбца):!:

:!:<wrap hi> 9-й и 10-й столбцы необходимы для шаблонов "ТД Кама завод - наши склады камеры ОПТ Ростов",  "ТД Кама завод - наши склады камеры ОПТ Новосиб".</wrap>

<wrap hi>**Ободные ленты заливаются в шаблон "ТД Кама завод".**</wrap>

В файле заливки для ободных лент указываем:
  * 1-й столбец - Артикул
  * 2-й столбец - Наименование
  * 3-й столбец - цена вход (из колонки "Минималь-ные цены на условиях отсрочки, руб. с НДС")
  * 4-й столбец - цена опт (из алгоритма)
  * 5-й столбец - цена розничная (аз алгоритма)
  * 6-й столбец - статус (под заказ)
