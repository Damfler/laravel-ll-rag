====== Starmaxx свободные остатки ======

Существует отдельный склад "Свободные остатки Starmaxx (Завод)" в котором находятся только остатки.

Обычное обновление.

Обновляем: остатки.

Категории товаров: грузовые шины, сх/сп шины.

В теме письма обычно не подписывают что именно пришло: прайс или остатки Starmaxx, поэтому лучше проверить все письма, которые присылают с темой письма "Starmaxx/стармакс". Файл в котором только остатки называется "PETLAS-STARMAXX DOM.STOCK" и выглядит следующим образом:

<wrap em>В данном файле есть остатки и для Starmaxx и для Petlas</wrap>

Для формирования наименования необходимо СЦЕПИТЬ следующие столбцы через пробел: "SIZE", "PR", "PATTERN", "TL/TT", "INDEX"

Остатки Starmaxx и Petlas заливаются одним файлом.

Для Petlas:

  * Артикул (CODE - красный)
  * Сформированное наименование
  * Пустой
  * Остаток (STOCK - PETLAS)

Для Starmaxx:

  * Артикул (CODE - синий)
  * Сформированное наименование
  * Пустой
  * Остаток (STOCK - Starmaxx)


Перед сопоставлением рекомендую просинхронизировать с заводом Starmaxx.
